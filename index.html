h1 até h6 (voltar)

Esses elementos representam níveis de cabeçalhos de seção.

Existem seis níveis: h1, h2, h3, h4, h5 e h6.

Em ordem crescente e tratando-se de tamanho da fonte, o maior nível é o "h1" e o menor é o "h6".

Podemos verificar isso nos exemplos abaixo:

p (voltar)

Elemento utilizado para demarcar um parágrafo.

Os textos que estiverem entre a tag de abertura e fechamento desse elemento, terão sua formatação inicial na forma de um parágrafo simples.

mark (voltar)

Esse elemento funciona como uma espécie de "marca texto". Ou seja, qualquer texto que estiver envolvido por essa tag terá uma cor de preenchimento.

Ex.: teste.

small (voltar)

Elemento que representa comentários laterais em tamanho de fonte menor do que as demais.

Ex.: fonte normal ==> Teste | fonte com elemento "small" ==> Teste.

i (voltar)

Consiste na formatação de um texto em itálico.

Ex.: Texto normal | Texto em itálico.

u (voltar)

Nada mais é do que o efeito sublinhado em um determinado trecho do texto.

Ex.: Texto normal | Texto sublinhado.

strong (voltar)

Elemento utilizado para deixar a fonte em negrito.

Ex.: Texto normal | Texto em negrito.

ol (voltar)

Esse elemento representa uma lista ordenada, cujos marcadores são dispostos em ordem.

Utilizada quando seus itens precisam seguir uma ordem específica.

Ex.:

  1. A
  2. B
  3. C

ul (voltar)

Consiste em uma lista não ordenada, cujos marcadores não representam uma ordem específica.

Geralmente usada quando a ordem dos itens não é importante.

Ex.:

  • C
  • B
  • A

li (voltar)

Tanto em uma lista ordenada quanto em uma lista não ordenada, esse elemento representa os seus itens.

Ex.:

Lista ordenada Lista não ordenada
  1. Item 1
  2. Item 2
  3. Item 3
  • Item 3
  • Item 1
  • Item 2

a (voltar)

Esse elemento cria um hiperlink para um lugar específico, definido pelo desenvolvedor, com o auxílio do atributo "href", seja para uma página na web, um arquivo em uma pasta local ou qualquer outra coisa cujo caminho esteja definido nesse atributo.

Um exemplo disso está em na opção "(voltar)", presente em cada seção dessa página ou até mesmo no sumário presente no início da página.

hr (voltar)

Essa tag cria uma linha horizontal no corpo da página.

Ela pode ser notada no final de cada seção dessa página.

sub (voltar)

Elemento que representa um texto subescrito.

Ex.: sem a tag ==> CO2 | com a tag ==> CO2

sup (voltar)

Enquanto o elemento "sub" deixa um texto subescrito, o "sup" faz o contrário, deixando o texto sobrescrito.

Ex.: sem a tag "sup" ==> a2 + b2 = c2 | com a tag "sup" ==> a2 + b2 = c2

blockquote (voltar)

O elemento blockquote indica que o texto dentro dessa tag é uma citação.

Um recuo nesse trecho se torna bem evidente.

É muito comum em livros e páginas web que citam trechos diversos. Veja o exemplo abaixo:

"Tu te tornas eternamente responsável por aquilo que cativas."

— O Pequeno Príncipe

font (voltar)

Atualmente, esse elemento se tornou obsoleto.

Trata-se de uma tag que possui a função de dar estilo ao texto, associada aos atributos 'color'(cor da fonte), 'face'(estilo da fonte a ser aplicada), 'size'(tamanho da fonte).

Esse tipo de tag não é mais recomendada, pois, essas estilizações se aplicam ao CSS.

del (voltar)

O elemento 'del' tem a função de destacar um trecho do texto que foi removido, destacando alterações.

Em contrapartida, existe o elemento 'ins' que faz a inserção de um trecho no texto. Confira no exemplo abaixo:

Texto removido ==> Desenvolvedor Web | Texto inserido ==> Web Developer

abbr (voltar)

Essa tag tem o objetivo de representar uma abreviação.

O elemento destaca essa abreviação e, associado ao atributo 'title', quando passamos o mouse sobre a parte destacada, podemos verificar do que se trata essa abreviação.

Ex.:Dio (Digital Innovation One)
